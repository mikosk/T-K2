1. Fyrir hvað stendur skammstöfunin RAM? Útskýrðu hvað RAM er.
	RAM stendur fyrir random acces memory . Vinnsluminni sem les gögn frá hörðum eða ssd diski yfir í móðurborðir á svakalega hraða.

2. Fyrir hvað stendur skammstöfunin ROM? Útskýrðu hvað ROM er.
	ROM stendur fyrir read-only memory. virkar eins og RAM nema það þarf ekki rafmagn til að halda "memory".
